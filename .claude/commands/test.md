# テスト実行

バックエンドテストの実行と結果確認:

## 実行手順

1. 全テスト実行:
```bash
cd backend && go test ./...
```

2. 特定パッケージのテスト:
```bash
go test ./internal/domain/tenant/...
```

3. 詳細出力:
```bash
go test -v ./...
```

4. カバレッジ確認:
```bash
go test -cover ./...
```

## 失敗時の対応

1. 失敗したテストを特定
2. エラーメッセージを分析
3. 関連するソースコードを確認
4. 修正を提案

## カバレッジ目標

- Domain層: 80%以上
- Usecase層: 70%以上
- Handler層: 60%以上

## チェックリスト

- [ ] 全テストがパス
- [ ] 新規コードにテストがある
- [ ] カバレッジが目標を満たす
